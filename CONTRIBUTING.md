# Contributing

Thanks for helping improve TREP.

## What to Contribute
- Clarity improvements
- Better prompts and questions
- Process refinements

## What Not to Contribute
- Generic startup advice
- Hustle or grind mindsets
- Anything that assumes I am the long-term operator

## How to Propose a Change
1. Open an issue describing the change
2. Explain the why, not just the what
3. If accepted, submit a PR

## Versioning and Commits

This project uses Semantic Versioning with tags like `vMAJOR.MINOR.PATCH`.

Please use Conventional Commits for changes:

- `feat: ...` for new capabilities
- `fix: ...` for bug fixes or corrections
- `docs: ...` for documentation changes
